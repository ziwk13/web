CSS

1. CSS(Cascading Style Sheets)는 웹 페이지의 디자인과 레이아웃을 담당하는 스타일 시트 언어입니다. 
2. HTML이 웹 문서의 구조와 내용을 정의한다면, CSS는 그 구조에 색상, 글꼴, 배치, 크기 등 시각적 요소를 추가해 웹 페이지의 외관을 꾸밉니다.
3. CSS의 주요 특징
  1) 디자인과 구조의 분리
    CSS를 사용하면 콘텐츠(HTML)와 디자인(CSS)을 분리하여 관리할 수 있습니다. 이로 인해 웹 문서의 유지보수와 관리가 쉬워지고, 여러 웹 페이지에 동일한 스타일을 쉽게 적용할 수 있습니다.
  2) 재사용성과 확장성
    한 번 정의한 CSS 파일을 여러 페이지에서 재사용할 수 있으며, 테마나 템플릿 형태로 확장도 가능합니다.
  3) 다양한 적용 방법
    CSS는 HTML 문서에 다음 세 가지 방식으로 적용할 수 있습니다.
    (1) 외부 스타일시트: 별도의 CSS 파일을 만들어 <link> 태그로 불러오기
    (2) 내부 스타일시트: HTML <head> 내 <style> 태그에 작성
    (3) 인라인 스타일: 개별 HTML 태그의 style 속성에 직접 작성
  4) 선택자와 선언부 구조
    CSS 규칙은 '선택자(selector)'와 '선언부(declaration)'로 구성됩니다. 선택자는 스타일을 적용할 HTML 요소를 지정하고, 선언부는 적용할 속성과 값을 정의합니다.
  5) 계단식(Cascading)과 우선순위
    CSS의 '계단식' 특성은 여러 스타일이 중첩될 때 우선순위에 따라 적용된다는 의미입니다. 
    인라인 스타일 > 내부 스타일시트 > 외부 스타일시트 순으로 우선순위가 높으며, !important 속성이 붙은 스타일이 가장 우선합니다.
  6) 반응형 디자인 지원
    CSS를 이용하면 PC, 태블릿, 스마트폰 등 다양한 기기에 맞춰 웹 페이지의 레이아웃을 유연하게 조정할 수 있는 반응형 웹을 구현할 수 있습니다.
  7) 박스 모델과 레이아웃 제어
    CSS는 박스 모델(콘텐츠, 패딩, 테두리, 마진) 기반으로 레이아웃을 제어하며, 요소의 크기와 위치를 세밀하게 조정할 수 있습니다.
  8) 웹 표준 준수
    CSS는 W3C(World Wide Web Consortium)에서 표준으로 관리되어, 웹의 일관성과 호환성을 높입니다.
4. CSS 문법
  1) 주석은 단일 형태입니다. /* */ 를 사용합니다.
  2) 기본 문법
    선택자(selector) {
      속성1: 값1;
      속성2: 값2;
      ...
    }
  3) 다양한 적용 방법
    (1) 외부 스타일시트
      별도의 .css 파일로 작성 후 <link> 태그로 연결합니다. 예) <link rel="stylesheet" href="style.css">
    (2) 내부 스타일시트
      <style> 태그에 스타일을 작성합니다. 예) <style> p { color: red; } </style>
    (3) 인라인 스타일
      태그에 직접 스타일을 작성합니다. 예) <p style="color: red;">문장</p>
